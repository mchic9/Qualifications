# Qualifications TP 2

1. Identifer les 3 erreurs 

Les 3 erreurs sont :
   La variable p n'existe pas dans le constructeur.
   Le constructeur ne reçoit aucun paramètre pour le prénom.
   Le nom n'est pas initialiser dans le constructeur.

2. Combien de parametres devrait avoir le constructeur ?

Il de devrait en avoir 3 le nom le prenom et numSecu.

3. Pourquoi il y a t-il un cadenas devant nom ?

Il y a un cadenas devant nom pour indiquer que c'est une variable privée, ce qui signifie qu'elle ne peut pas être accédée directement depuis l'extérieur de la classe.

5. Quels sont les classes qui sont en liaison avec la classe Employe ?

La classe Qualification est en liaison avec la classe Employe, car un employé peut avoir une qualification.
